# Changelog

Все значимые изменения в документации фиксируются на этой странице.

## 2026-04-30

### Added

- Добавлен полноценный раздел `UX-портфолио` под резюме `UX Writer / Product Writer`.
- Добавлены UX-кейсы: онбординг нового пользователя, аудит формы и чеклист UX-текста.
- Главная страница обновлена под двойное позиционирование: `Технический писатель / DocOps / UX Writer`.

### Changed

- Раздел `UX-писательство` переработан в витрину UX-кейсов с описанием процесса работы.
- Страница `Обо мне и контакты` обновлена под две версии профиля: technical documentation и UX writing.

### Changed

- Ссылка на GitHub-репозиторий обновлена после переименования проекта в `portfolio`.

## 2026-04-28

### Added

- Для страницы `Changelog` добавлен front matter с `ShowToc: false`, чтобы убрать шумный TOC в боковой панели.

### Changed

- Обновлена палитра портала: базовый фон сменен на `#1c1e20` для более мягкого тёмного тона.
- Главная страница адаптирована под Hugo/PaperMod: заменен MkDocs-синтаксис кнопок на HTML-кнопки (`cta-btn`), исправлены ссылки карточек.
- Переписаны кастомные стили под PaperMod, убраны конфликтующие правила `.md-*` от MkDocs.
- Для контейнера `.main` снято принудительное растягивание (`min-height`), чтобы не создавать пустое пространство на коротких страницах.

### Fixed

- Исправлена читаемость `Interactive API`: повышен контраст текста и элементов в `Swagger UI` на тёмной теме.
- Дополнительно исправлена читаемость блока `Schemas` (модели, свойства, скобки и разделители).
- Убран фон у `pre`-блоков (`.post-content pre`) для более чистого отображения кода.

## 2026-04-19

### Added

- Добавлена страница `Interactive API` со Swagger UI на базе `openapi/openapi.yaml`.
- Миграция портала на тему `Material for MkDocs`.
- Добавлены разделы портала: `Бизнес-аналитика`, `Разработчикам`, `Пользователям`, `О проекте`.
- Добавлены страницы с контентом: BPMN, C4, UML, onboarding, troubleshooting, дашборд продаж.
- Добавлены исходники диаграмм `PlantUML` (`docs/diagrams/*.puml`) и скрипт рендера `scripts/render-diagrams.sh`.
- Добавлена страница `Docs as Code` с описанием подхода.

### Changed

- Репозиторий и сайт переведены на новое имя: `petstore-api-docs` (обновлен `site_url`).
- Усилен `Quick Start`: добавлен пример на JavaScript (`fetch`) и блок проверки типовых ошибок (`400`, `404`).
- Обновлена навигация портала с многоуровневой структурой.
- CI workflow обновлен: установка зависимостей из `requirements.txt`, рендер PlantUML-диаграмм и сборка портала перед деплоем.
- Добавлены критерии качества и измеримые показатели на главной странице.
- Добавлены разделы обратной связи на ключевые страницы портала.
- Добавлен `README.md` для быстрого ревью кейса рекрутером/лидом.
- Добавлен финальный `QA Checklist` в документацию.
- В CI добавлена валидация OpenAPI спецификации перед рендером и деплоем.
- Улучшен UX портала: обновлены тема/палитра, вкладки навигации, breadcrumbs и интегрированный TOC.
- Главная страница переработана в формат карточек разделов для быстрого сканирования.
- Удалены страницы-заглушки из структуры документации.

### Fixed

- Локализация пагинации (`Previous`/`Next`) стабилизирована с учетом динамической перерисовки темы.
- Исправлен CSS-отступ заголовков (`margin-bottom: 10px`) через `docs/stylesheets/extra.css`.
- Исправлена загрузка OpenAPI в `Interactive API` (fallback путей, устранен `404`).
- Исправлены несовместимые диаграммы `xychart-beta` в сценариях (замена на совместимый формат).

### QA

- Release check passed: проверены ключевые страницы `latest/` и root (`/`), включая `Interactive API`, сценарии, версии и changelog.
